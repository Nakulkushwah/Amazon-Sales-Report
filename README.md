# Amazon Sales Report – Power BI Analysis

## Overview
This project provides a detailed analysis of Amazon sales data using Power BI. The dataset includes order details such as order ID, product category, size, quantity, amount, fulfillment method, order status, shipping details, and customer location.
The objective is to uncover trends, patterns, and actionable insights that support data-driven decision-making and business improvement.

## Objectives
-Analyze overall sales performance and trends over time
-Identify top-selling products, categories, and sizes
-Compare fulfillment methods (MFN vs AFN)
-Segment customers based on buying behavior and location
-Explore geographic sales distribution
-Provide recommendations to improve sales, inventory, and customer service

## Dashboard Features
-Sales Overview & KPIs
-Monthly/Yearly Sales Trends
-Category & Size-wise Product Analysis
-Fulfillment Performance Comparison
-ustomer Segmentation
-State & City-wise Sales Map
-Profitability & Order Status Insights

## Tools & Technologies
Power BI – Data modeling & visualizations
DAX – Measures & calculations
Excel/CSV – Data source

## Key Insights
-Identified high-revenue product categories
-Highlighted top-performing regions
-Observed seasonal variations in sales
-Found patterns in customer purchase frequency
-Evaluated fulfillment efficiency
